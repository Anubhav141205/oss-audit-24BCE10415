## 📜 Project Overview
This project analyzes Git as an open-source software, focusing on its origin, licensing, Linux footprint, ecosystem, and comparison with proprietary alternatives. It also includes five shell scripts demonstrating Linux and scripting concepts.

---

## ⚙️ Shell Scripts

### 1. System Identity Report (`script1.sh`)
Displays system information such as kernel version, user, uptime, and OS details.

### 2. FOSS Package Inspector (`script2.sh`)
Checks whether Git is installed and displays its version and details.

### 3. Disk and Permission Auditor (`script3.sh`)
Analyzes system directories, showing their size, permissions, and ownership.

### 4. Log File Analyzer (`script4.sh`)
Reads a log file and counts occurrences of a keyword like "error".

### 5. Open Source Manifesto Generator (`script5.sh`)
Generates a personalized open-source philosophy statement based on user input.
